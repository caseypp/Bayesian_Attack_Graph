Class CIDPotentials

Class CIDPotential implements basic operations with factors. This class is an abstract one, so it can’t be used itself.

Operations that can be performed with a CIDPotential object are as follows:

— combination and division of CIDPotential object

— contraction of  CIDPotential objects

— marginalization

 

Combine

Combines two factors and returns product.
virtual CIDPotential* Combine(const CIDPotential *pOtherFactor,

    CIDPotential* output = NULL) const;

Arguments
pOtherFactor Pointer to the multiplier factor
output If this argument is NULL, function creates a new CIDPotential object, else it only reloads factor.
Discussion

This function returns pointer to a CIDPotential object, which is the product of two

CIDPotential objects.

Contraction

Multiplies utility and probability parts of factor and returns the product

virtual CDenseMatrix<float>* Contraction(CDenseMatrix<float>* output = NULL) const;

Arguments
output If this argument is NULL, function creates a new CDenseMatrix object and fills it with the result of contraction.
Discussion
This function returns pointer to a CDenseMatrix object, which is the product of contraction  between utility and probability parts.

DivideInSelf

Provides division and puts the result to potential.

virtual void DivideInSelf(const CCPD *pOtherFactor) const;
Arguments
pOtherFactor Pointer to the divider factor
Discussion

This function changes the object, for which it was called, by changing its data

matrices.

InsertPoliticsInSelf

Inserts new politics to the current CIDPotential object

virtual void InsertPoliticsInSelf(const CCPD *pOtherFactor) const;
Arguments
pOtherFactorPointer to the factor, which contains new politics
Discussion
This function inserts new politics, which are described by argument, to current object

SetProbDistribFun

Sets the probability part of potential

void SetProbDistribFun(const CDistribFun* data);

Arguments
data Pointer to a CDistribFun object, which contains probability part of potential.
Discussion

This function changes the object, for which it was called, by changing its data

matrices.

SetUtilityDistribFun

Sets the utility part of potential.

void SetUtilityDistribFun(const CDistribFun* data);
Arguments
data Pointer to a CDistribFun object, which contains probability part of potential.
Discussion

This function changes the object, for which it was called, by changing its data

matrices.
